Xpell Platform

Xpell is a layered runtime platform for AI-native systems. This page is a high-level map of the core parts, not deep package documentation.

Layered Architecture

@xpell/core

XData 2, Nano-Commands 2, XEM, XModule, and XObject primitives for deterministic runtime behavior.

@xpell/node

Xpell Server, Wormholes 2 transport, and XDB-backed persistence for server-side runtime execution.

@xpell/ui

XUI and XUIObject composition with a Wormholes client boundary for real-time interface updates.

@xpell/3d

Three.js-based 3D layer for structured scene composition and consistent command boundaries.
